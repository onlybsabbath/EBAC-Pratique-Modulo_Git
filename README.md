Uma Calculadora Python que executa as quatro operações algébricas elementares, além da divisão inteira, resto da divisão inteira e exponenciação.

Funcionamento da Calculadora Python:
  - Inserir o primeiro número (num1)
  - Inserir o segundo número (num2)
  - A Calculadora retornará o resultado da soma, subtração, multiplicação, divisão, divisão inteira, resto da divisão inteira e exponenciação
    - Soma: num1 + num2
    - Subtração: num1 - num2
    - Multiplicação: num1 * num2
     - Se num=2 é diferente de 0, então:
        - Divisão: num1 / num2
        -  Divisão inteira: num1 // num2
        -  Resto da divisão inteira: num1 % num2
    - Exponenciação: num1 ** num2

Obs: Para executar um arquivo com extensão .sh, utilize o comando "./nome_arquivo.sh" (ex: ./calculadora.sh)
